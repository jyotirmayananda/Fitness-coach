# **App Name**: AI Fitness Fusion

## Core Features:

- Personalized Plan Generation: Generate workout and diet plans tailored to user details using an LLM tool. Details include name, age, gender, fitness goals, current level, location and diet.
- AI-Powered Tips: Provide AI-generated fitness and lifestyle tips.
- Voice Plan Reading: Enable users to listen to their workout and diet plans using a text-to-speech API.
- Exercise Image Generation: Generate images of exercises and meal items using an AI image generation API when the user clicks on those entries.
- Plan Export: Allow users to export their generated plan as a PDF document.
- Theme Toggle: Allow users to switch between light and dark mode.
- Plan Regeneration: Allow the user to regenerate the plan in case they want a different suggestion

## Style Guidelines:

- Primary color: Forest green (#38A3A5) to evoke health and growth.
- Background color: Light beige (#F5F5DC), very desaturated, for a clean and natural feel.
- Accent color: Pale salmon (#F47264), a warm and inviting contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern, readable feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, consistent icons from a set like Material Design Icons.
- Smooth transitions and subtle animations with Framer Motion.
